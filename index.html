Como usar o site

Este guia resume o fluxo de cada visão, os atalhos principais e as ações de importação, exportação e organização dos apontamentos.

Navegação geral

Use os botões do topo para alternar entre as visões do sistema. Os atalhos são: C para Calendário, T para Tabela, G para Gantt, R para Gráficos e D para Diretório.

  • Sem. anterior, Hoje e Prox. semana mudam o período exibido.
  • O botão com sol/lua alterna entre modo claro e escuro.
  • O botão de configurações reúne a ajuda desta tela e as ações de limpeza.

Visão Calendário

Esta é a visão principal para criar e organizar apontamentos por dia e horário.

  • Para criar um apontamento, clique ou arraste em um espaço vazio da grade. A faixa destacada mostra o intervalo que será usado antes de abrir o popup.
  • Ao passar o mouse em um horário vazio, aparece uma sombra leve para facilitar a leitura do ponto exato da grade.
  • Para editar, clique em um evento existente. Para mover, arraste o bloco para outro horário.
  • Para ajustar a duração, arraste a borda superior ou inferior do evento.
  • Durante o arraste ou redimensionamento, um bloco suave marca a posição original do evento.
  • Alterações de horário no mesmo dia são aplicadas direto. A confirmação só aparece quando o evento muda para outra data.

Popup do apontamento

O popup permite preencher ou revisar todos os dados do evento antes de salvar.

  • Defina data, início e fim em intervalos de 10 minutos.
  • Informe o projeto, escolha ou digite a atividade e confira o ID Artia. Quando a base estiver carregada, o sistema tenta preencher o ID automaticamente.
  • Marque Lançamento Artia quando o registro já tiver sido lançado no sistema externo.
  • Use os botões de Local de trabalho para inserir o marcador no campo de observação.
  • Copiar e Colar reaproveitam projeto, atividade, ID e observação entre eventos.
  • Salvar grava o apontamento. Cancelar fecha sem aplicar mudanças. Atalhos: Ctrl + Enter para salvar e Esc para cancelar.

Visão Tabela

Mostra os apontamentos em formato tabular para revisão rápida, filtragem e edição pontual.

  • Use os filtros de período, projeto e atividade para refinar os resultados.
  • Clique nos cabeçalhos das colunas para ordenar os dados.
  • O botão + Novo apontamento abre o mesmo popup de criação usado no calendário.
  • Clique em uma linha para abrir o apontamento em modo de edição.
  • A tabela mostra horários, esforço, observação, status de lançamento no Artia e ID da atividade.

Visão Gantt

Resume a semana por projeto, com total de horas e distribuição por dia.

  • Use o filtro de Projeto para isolar um item específico.
  • A primeira coluna identifica o projeto, a segunda mostra o total acumulado na semana e as demais colunas distribuem as horas por dia.
  • Esta visão é útil para comparar carga semanal entre projetos sem abrir os eventos individualmente.

Visão Gráficos

Permite analisar volume de horas por período e por projeto.

  • Defina o período, o projeto e o agrupamento por Meses ou Semanas.
  • O gráfico Horas (ao Longo do Tempo) combina barras por período com a linha de horas acumuladas.
  • O gráfico Projetos (Distribuição de Horas) mostra a participação de cada projeto e exibe detalhes ao passar o mouse.

Visão Diretório

Centraliza a base de IDs carregada por projeto e atividade.

  • Use a busca para localizar um número de projeto ou parte do nome de uma atividade.
  • O botão de ordenação alterna entre lista crescente e decrescente dos projetos.
  • Ao selecionar um projeto, a coluna da direita lista as atividades disponíveis e seus respectivos IDs.

Importação, exportação e dados

Os botões da área superior direita cuidam da entrada e saída dos dados do site.

  • E-mail: define o e-mail incluído na exportação CSV.
  • Exportar CSV: gera um CSV com os apontamentos visíveis para uso externo.
  • Importar Apontamentos (XLSX): lê uma planilha e adiciona os registros ao sistema.
  • Definir Local de Salvamento: escolhe a pasta usada para backups XLSX.
  • Salvar XLSX: gera o backup manual dos apontamentos.
  • Carregar IDs Artia (XLSX): importa a base usada para preencher e consultar IDs por projeto e atividade.
